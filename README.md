## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript repository.

TenPick 서비스의 백앤드 구성 프로젝트

## DB 환경 구성

DB : mariadb 10.6

Docker 디렉토리 참조

## 환경 설정 

.local.evn 파일 : 로컬 PC에서 개발 환경 시 사용

.test.env 파일 : 테스트 서버 환경 시 사용

.production.evn 파일 : 실서버 환경 시 사용 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
